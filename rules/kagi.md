## Searching the web

If you need to search the web, you must use the `kagi-ken-cli` CLI tool. It is already configured and ready to go.

```bash
kagi-ken-cli search "some example query"
```

## Summarizing web pages

When asked to summarize a URL, you must use the `kagi-ken-cli` CLI tool. It is already configured and ready to go. Example:

```bash
kagi-ken-cli summarize --url https://example.com/
```
